# init-ayx-notes

## 简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面涵盖不同主题和用途，作为个人笔记和项目记录的静态展示。

所有页面均为独立 HTML 文件，无需后端支持，可直接在浏览器中打开或通过 GitHub Pages 访问。

## 目录说明

```
├── README.md           # 本文件
├── index.html          # 仓库首页/导航页
├── pages/              # 存放各个 HTML 页面的目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立 HTML 页面存放于此，每个文件为一个独立页面。
- `assets/`：页面引用的公共资源，包括样式、脚本和图片等。

## 页面归档说明

- 每个页面均为独立、自包含的 HTML 文件。
- 页面之间无强依赖关系，可单独引用或部署。
- 页面内容不定期更新或新增，不保证所有页面均保持最新状态。
- 部分页面可能包含外部资源链接（如字体、库文件），这些资源由第三方提供，仓库不保证其可用性。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交问题或建议，但不对响应时间作承诺。
- 不接受直接向仓库提交修改类的 Pull Request（除非明显错误），但欢迎讨论。

本仓库内容仅供个人学习与参考。如需引用或再发布，请保留原始出处说明。
